Covid-19 Centroamerica

2020-04-09

Esta es una gráfica que muestra la evolución de caso de Covid-19 en los países de Centroamérica. Muestra la tasa de contagios y el pronóstico de en cuantos días se podría doblar el número de caso. Para más detalle cómo se tiene la información ver: https://robjhyndman.com/hyndsight/logratios-covid19/

library(devtools)
library(tidyverse)
library(tsibble)
library(tidycovid19)
updates <- download_merged_data(cached = TRUE)

#countries <- c("AUS", "BRA", "ITA", "ESP", "USA","GBR")
countries <- c("CRI", "HON", "GTM", "SLV", "PAN")
updates %>%
  mutate(
    cases_logratio = difference(log(confirmed))
  ) %>%
  filter(iso3c %in% countries) %>%
  filter(date >= as.Date("2020-03-01")) %>%
  ggplot(aes(x = date, y = cases_logratio, col = country)) +
   labs(title = "Comportamiento de Casos Covid-19 paises de Centroamerica",
       caption = "Fuente: Johns Hopkins University Center for Systems Science and Engineering")+
  geom_hline(yintercept = log(2)/c(2:7,14,21), col='grey') +
  geom_smooth(method = "loess", se = FALSE) +
  scale_y_continuous(
    "Incremento diario de casos",
    breaks = log(1+seq(0,60,by=10)/100),
    labels = paste0(seq(0,60,by=10),"%"),
    minor_breaks=NULL,
    sec.axis = sec_axis(~ log(2)/(.),
                        breaks = c(2:7,14,21),
                        name = "Dias para doblar casos")
  )+
  scale_x_date(date_labels = "%b %d",limits = as.Date(c("2020-03-15", "2020-04-11")),
                    breaks = function(x) seq.Date(from = as.Date("2020-03-01"), to = as.Date("2020-04-11"),by = "4 days")
)
